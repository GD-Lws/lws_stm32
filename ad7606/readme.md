## 管脚连接
|   STM32    | AD7706  |  功能  |
|:-----------|--------:|:-----:|
|    E15     | CV      | 启动 |
|    E13     | RST     | 硬件复位 |
|    B10     | RAGE    | 量程确定 |
|    E11     | CS      | 片选 |
|    E10     | BUSY    | 忙碌信号 |
|    A5      | SCK/RD  | 时钟信号 |
|    A6      | MISO/DB7| 输入信号 |
|    GND     | OS0     | 过采样模式引脚 |
|    GND     | OS1     | 过采样模式引脚 |
|    GND     | OS2     | 过采样模式引脚 |
|   5V/3.3   | VIN     | 逻辑电源输入，决定逻辑接口工作电压	|
|    GND     | DB15    | 选择为SPI模式 |
